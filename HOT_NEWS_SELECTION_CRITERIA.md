# Hot/인기 뉴스 선정 기준 및 방법

**작성일**: 2026년 1월 15일

---

## 📊 Hot 뉴스 선정 개요

Hot 뉴스는 **포트폴리오와 무관하게** 해외시장과 국내시장에서 현재 가장 주목받는 뉴스를 수집합니다. 시장 전반의 흐름과 심리를 파악하는 데 활용됩니다.

---

## 🌎 해외시장 Hot 뉴스 선정 기준

### 수집 소스: RSS 피드 (Bloomberg, CNBC, Investing.com) + Yahoo Finance

**방법**: RSS 피드 우선, 실패 시 yfinance fallback

### 우선순위

1. **RSS 피드** (feedparser 사용 가능 시):
   - Bloomberg: `https://feeds.bloomberg.com/markets/news.rss`
   - CNBC: `https://www.cnbc.com/id/100003114/device/rss/rss.html`
   - Investing.com: `https://www.investing.com/rss/news.rss`
   - Yahoo Finance RSS: `https://feeds.finance.yahoo.com/rss/2.0/headline`

2. **Fallback: yfinance** (RSS 실패 시):
   - 다양한 시장 지수와 섹터 티커 사용 (포트폴리오 무관)
   - 티커: `^GSPC`, `^DJI`, `^IXIC`, `^VIX`, `^TNX`, `GC=F`, `CL=F` 등
   - 각 티커에서 최신 뉴스 수집

### 선정 방법

1. **뉴스 수집**: RSS 피드 또는 yfinance를 통해 최신 뉴스 수집
2. **정렬 기준**: **최신순** (소스가 제공하는 순서대로)
3. **필터링**: **없음** (포트폴리오 관련성 무관, 현재 화두가 되는 뉴스 수집)
4. **중복 제거**: 제목 기준으로 중복 제거
5. **최종 선택**: 상위 **10개** 선택

### 수집 정보

- 제목 (Title)
- 요약문 (Summary, 최대 200자)
- 링크 (Link)
- 출판사 (Publisher)

## 🇰🇷 국내시장 Hot 뉴스 선정 기준

### 수집 소스: 네이버 금융

**방법**: HTML 크롤링 (BeautifulSoup)

**URL**: `https://finance.naver.com/news/news_list.naver?mode=LSS2D&section_id=101&section_id2=258`

- `section_id=101`: 증권/투자 섹션
- `section_id2=258`: 시황/종목 분석 섹션

### 선정 방법

1. **뉴스 수집**: 네이버 금융 뉴스 리스트 페이지에서 HTML 크롤링
2. **정렬 기준**: **페이지 상단 순서** (네이버가 제공하는 순서대로, 일반적으로 최신순/인기순 혼합)
3. **필터링**: **없음** (포트폴리오 관련성 무관, 현재 화두가 되는 뉴스 수집)
4. **품질 필터**: 제목 길이 10자 이상만 포함
5. **최종 선택**: 상위 **10개** 선택

### 수집 정보

- 제목 (Title)
- 요약문 (Summary, 최대 200자)
- 링크 (Link)

## 🔍 선정 기준 요약

### 공통 기준

| 항목 | 기준 |
|------|------|
| **포트폴리오 필터링** | ❌ 없음 (무관한 뉴스도 포함) |
| **정렬 기준** | 최신순 (소스가 제공하는 순서) |
| **중복 제거** | ✅ 제목 기준 |
| **최종 개수** | 해외 10개 + 국내 10개 = 총 20개 |

### 해외시장 특화 기준

- **대상**: RSS 피드 우선 (Bloomberg, CNBC, Investing.com, Yahoo Finance)
- **Fallback**: RSS 실패 시 다양한 시장 지수 티커 사용 (^GSPC, ^DJI, ^IXIC, ^VIX, ^TNX, GC=F, CL=F 등)
- **수집량**: 상위 10개 선택
- **목적**: 포트폴리오와 무관하지만 현재 화두가 되고 있는 뉴스 수집

### 국내시장 특화 기준

- **대상**: 네이버 금융 시황/종목 분석 섹션
- **품질 필터**: 제목 길이 10자 이상
- **수집량**: 페이지에서 상위 10개 선택
- **목적**: 포트폴리오와 무관하지만 현재 화두가 되고 있는 뉴스 수집

## 📈 Hot 뉴스 vs 포트폴리오 관련 뉴스 비교

### 포트폴리오 관련 뉴스 (`get_market_news_with_context`)

- **필터링**: ✅ Smart Filter 적용 (포트폴리오 종목 +5점, 시장 키워드 +2점)
- **목적**: 내 포트폴리오와 직접 관련된 뉴스
- **개수**: 10개
- **선정 기준**: 가중치 점수 기반

### Hot/인기 뉴스 (`get_hot_news`)

- **필터링**: ❌ 없음
- **목적**: 시장 전반의 Hot/인기 뉴스 (포트폴리오 무관)
- **개수**: 해외 5개 + 국내 5개 = 10개
- **선정 기준**: 최신순 (소스 제공 순서)

## 🎯 Hot 뉴스 활용 목적

1. **시장 전반 흐름 파악**: 포트폴리오와 무관하더라도 시장 전체의 동향 파악
2. **섹터 로테이션 감지**: 어떤 섹터가 주목받는지 파악
3. **거시경제 동향**: 시장 전반에 영향을 미치는 거시경제 뉴스 파악
4. **심리 파악**: 시장 참여자들의 관심사와 심리 파악

## ⚠️ 제한사항

### 해외시장

- Yahoo Finance API의 뉴스 순서에 의존
- 특정 티커의 뉴스가 많으면 해당 티커 뉴스가 과도하게 포함될 수 있음
- "인기" 기준이 명시적이지 않음 (최신순 기반)

### 국내시장

- 네이버 금융 페이지 구조 변경 시 크롤링 실패 가능
- 페이지 순서가 "최신순"인지 "인기순"인지 명확하지 않음
- 네트워크 오류 시 수집 실패

---

## 🔧 개선 가능 사항

1. **인기도 지표 활용**: 조회수, 댓글 수 등 인기도 지표 활용
2. **다양한 소스**: Bloomberg, Reuters 등 추가 소스 활용
3. **시간 가중치**: 최근 24시간 내 뉴스에 가중치 부여
4. **카테고리 필터**: 특정 카테고리(정치, 경제, 기업 등) 필터링 옵션

---

**참고**: 현재 구현은 "최신순" 기반이며, 소스가 제공하는 순서를 그대로 따릅니다. 명시적인 "인기도" 지표는 사용하지 않습니다.
