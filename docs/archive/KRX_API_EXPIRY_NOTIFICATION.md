# KRX API 키 유효기간 종료 알림 기능

## 구현 내용

KRX API 키의 유효기간이 종료되거나 만료 임박 시 사용자에게 텔레그램 메시지로 알림을 제공하는 기능을 구현했습니다.

## 구현된 기능

### 1. 유효기간 정보 저장
- **위치**: `config/settings.py`
- **환경변수**: `KRX_API_KEY_EXPIRY` (형식: `YYYY-MM-DD`, 예: `2027-01-12`)
- **특징**: Optional (설정하지 않아도 기존 기능 정상 작동)

### 2. 유효기간 체크 로직
- **위치**: `src/crawler.py`의 `get_kr_stock_data_krx_api()` 함수
- **동작**:
  - KRX API 호출 시 401 오류 발생 시 유효기간 체크
  - 만료일과 현재 날짜 비교
  - 전역 변수로 상태 추적 (`_krx_api_expired`, `_krx_api_expiry_checked`)

### 3. 상태 확인 함수
- **함수명**: `get_krx_api_status()`
- **반환값**:
  ```python
  {
      'expired': bool,              # 유효기간 만료 여부
      'expiry_date': date or None,  # 만료일
      'days_until_expiry': int or None  # 만료까지 남은 일수
  }
  ```

### 4. 텔레그램 경고 메시지
- **위치**: `src/main.py`의 `main()` 함수
- **발송 시점**: 바리케이트 메시지 직후
- **발송 조건**:
  - 유효기간 만료: 즉시 경고 메시지 발송
  - 7일 이내 만료 예정: 만료 임박 경고 메시지 발송

## 경고 메시지 형식

### 401 오류 발생 시 (유효기간 정보 없음)
```
⚠️ KRX API 키 인증 실패 안내

KRX API 호출 시 인증 오류가 발생했습니다.
가능한 원인:
• 인증키 유효기간 만료
• API 이용 신청 미승인
• 인증키 오류

현재는 네이버 크롤링으로 데이터를 수집하고 있습니다.
KRX Data Marketplace에서 인증키 상태를 확인하고 갱신해 주세요.

🔗 https://openapi.krx.co.kr
```

### 유효기간 만료 시 (유효기간 정보 있음)
```
⚠️ KRX API 키 유효기간 만료 안내

KRX API 키의 유효기간이 만료되었습니다.
만료일: 2026년 01월 13일

현재는 네이버 크롤링으로 데이터를 수집하고 있습니다.
KRX Data Marketplace에서 새로운 인증키를 발급받아 주세요.

🔗 https://openapi.krx.co.kr
```

### 만료 임박 시 (7일 이내)
```
⚠️ KRX API 키 유효기간 만료 임박 안내

KRX API 키의 유효기간이 곧 만료됩니다.
만료일: 2027년 01월 12일
남은 일수: 5일

만료 전에 KRX Data Marketplace에서 인증키를 갱신해 주세요.

🔗 https://openapi.krx.co.kr
```

## 설정 방법

### .env 파일에 추가
```bash
# KRX API 인증키
KRX_API_KEY=your_api_key_here

# KRX API 유효기간 (선택사항, YYYY-MM-DD 형식)
KRX_API_KEY_EXPIRY=2027-01-12
```

### GitHub Secrets에 추가 (GitHub Actions 사용 시)
- `KRX_API_KEY`: 인증키
- `KRX_API_KEY_EXPIRY`: 유효기간 (예: `2027-01-12`)

## 동작 방식

### 1. 정상 작동 시
- 유효기간이 정상이거나 설정되지 않은 경우
- KRX API 호출이 정상적으로 작동하는 경우
- 경고 메시지 발송 안 함
- 기존 기능 정상 작동

### 2. 401 오류 발생 시 (유효기간 정보 없음)
- KRX API 호출 시 401 오류 발생
- 유효기간 정보가 없어도 만료로 간주
- 일반 경고 메시지 발송:
  - "⚠️ KRX API 키 인증 실패 안내"
  - 가능한 원인 안내 (만료, 미승인, 오류)
  - 네이버 크롤링으로 대체 중임을 안내

### 3. 만료 임박 시 (7일 이내, 유효기간 정보 있음)
- 매 리포트 생성 시마다 경고 메시지 발송
- 만료일과 남은 일수 표시
- 인증키 갱신 안내

### 4. 만료 시 (유효기간 정보 있음)
- 매 리포트 생성 시마다 경고 메시지 발송
- 만료일 표시
- 네이버 크롤링으로 대체 중임을 안내
- 새 인증키 발급 안내

## 주의사항

1. **유효기간 정보는 선택사항**
   - 설정하지 않아도 기능 정상 작동
   - **401 오류 발생 시 자동으로 만료로 간주하여 경고 메시지 발송**
   - 유효기간 정보가 있으면 상세 메시지, 없으면 일반 메시지 발송

2. **날짜 형식**
   - 반드시 `YYYY-MM-DD` 형식 사용
   - 예: `2027-01-12` (O), `2027/01/12` (X)

3. **자동 갱신 없음**
   - 인증키는 수동으로 갱신해야 함
   - 경고 메시지는 알림 목적만 제공
