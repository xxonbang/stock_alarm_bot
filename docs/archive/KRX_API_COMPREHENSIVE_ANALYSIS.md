## 1. 추가 API 신청 필요 여부 분석

### 현재 신청 완료한 API
1. ✅ **유가증권 일별매매정보**
2. ✅ **ETF 일별매매정보**

### 필요한 데이터 vs 제공 API 매핑

#### A. 외국인/기관 순매매량
- **필요 데이터**: 
  - 외국인 순매매량 (만 주, 최근 3거래일 합계)
  - 기관 순매매량 (만 주, 최근 3거래일 합계)

- **제공 API**: ✅ **유가증권 일별매매정보**
  - 컬럼명 확인됨:
    - `FRGN_NTBY_QTY`: 외국인 순매수 수량
    - `ORG_NTBY_QTY`: 기관 순매수 수량
  - 또는 한글 컬럼명: `외국인순매수`, `기관순매수`

- **결론**: ✅ **추가 API 불필요** (유가증권 일별매매정보에 포함됨)

#### B. ETF 괴리율
- **필요 데이터**: 
  - ETF 괴리율 (NAV 대비 %)

- **제공 API**: ✅ **ETF 일별매매정보**
  - NAV 및 시장가 정보 포함 예상
  - 괴리율 계산 가능 (시장가 - NAV) / NAV * 100

- **결론**: ✅ **추가 API 불필요** (ETF 일별매매정보에 포함될 가능성 높음)

### 포트폴리오 종목 분석

**현재 포트폴리오 구성:**
- 전체 종목: 24개
- 코스피 종목 (.KS): 11개 ✅
- 코스닥 종목 (.KQ): 0개 ✅

**결론:**
- ✅ 코스닥 종목 없음
- ✅ "코스닥 일별매매정보" API **불필요**

### 추가로 고려할 수 있는 API (선택사항)

#### 1. 코스닥 일별매매정보
- **필요성**: ⭐ (현재 포트폴리오에 코스닥 종목 없음)
- **현재**: 미신청
- **권장**: ❌ **불필요** (코스닥 종목이 추가될 경우에만 신청)

#### 2. 지수 정보 API
- **필요성**: ⭐ (시장 전체 흐름 파악용, 현재 프로젝트에서 필수 아님)
- **현재**: 미신청
- **권장**: ❌ **선택사항** (현재 프로젝트에서 필수 아님)

### 최종 권장사항

**✅ 현재 신청한 2개 API로 충분함**
1. ✅ **유가증권 일별매매정보**

2. ✅ **ETF 일별매매정보**

**❌ 추가 API 신청 불필요**

## 3. API 응답 구조 예상

### 유가증권 일별매매정보 API

**엔드포인트:**
```
URL: https://data-dbg.krx.co.kr/svc/apis/sto/stk_bydd_trd
Method: GET
Headers: {
    "AUTH_KEY": "발급받은_인증키"
}
Parameters: {
    "basDd": "YYYYMMDD",  # 기준일자 (필수)
    "isuCd": "종목코드"    # 종목코드 (선택, 전체 조회 시 생략)
}
```

**예상 응답 구조:**
```json
{
    "OutBlock_1": [
        {
            "ISU_CD": "005930",           // 종목코드
            "ISU_NM": "삼성전자",         // 종목명
            "MKT_NM": "KOSPI",            // 시장구분
            "TDD_CLSPRC": "70000",        // 종가
            "ACC_TRDVOL": "15000000",     // 거래량
            "ACC_TRDVAL": "1050000000",   // 거래대금
            "FRGN_NTBY_QTY": "1000000",   // 외국인 순매수 수량 ⭐
            "ORG_NTBY_QTY": "500000",     // 기관 순매수 수량 ⭐
            // 또는 한글 컬럼명:
            // "외국인순매수": "1000000",
            // "기관순매수": "500000"
        }
    ]
}
```

### ETF 일별매매정보 API

## 4. 구현 계획

### 승인 후 구현 필요 사항

#### A. 유가증권 일별매매정보 파싱 로직
```python
# 응답에서 외국인/기관 순매매량 추출
if 'OutBlock_1' in data:
    for row in data['OutBlock_1']:
        # 컬럼명 확인 후 추출
        foreign_qty = row.get('FRGN_NTBY_QTY') or row.get('외국인순매수', 0)
        org_qty = row.get('ORG_NTBY_QTY') or row.get('기관순매수', 0)
        
        # 만주로 변환
        foreign_net += float(foreign_qty) / 10000
        institutional_net += float(org_qty) / 10000
```

#### B. ETF 일별매매정보 파싱 로직
```python
# 응답에서 NAV 및 시장가 추출
if 'OutBlock_1' in data:
    for row in data['OutBlock_1']:
        market_price = float(row.get('TDD_CLSPRC', 0))
        nav = float(row.get('NAV', 0))
        
        if nav > 0:
            disparity_rate = ((market_price - nav) / nav) * 100
```
