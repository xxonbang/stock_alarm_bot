# 텔레그램 투자 리포트 v3: 근본적 재설계

## 이전 설계의 문제

v1(데이터 나열)과 v2(의사결정 대시보드)는 둘 다 **"더 좋은 데이터를 더 잘 보여주자"**에 머물렀다.
행동재무학 연구는 명확하다: **정보를 추가할수록 의사결정 품질은 떨어진다.**

> "모든 데이터 포인트에 질문하라: 이것이 오늘 의사결정을 바꾸는가? 아니면 그냥 재미있는가?"

---

## 핵심 발견: 이 포트폴리오의 진실

### 7개 ETF가 아니라 1개의 베팅이다

```
┌─────────────────────────────────────────────────┐
│  🔴 테크 블록 (~85%)                              │
│                                                   │
│  KODEX 나스닥100 ──┐                              │
│  KODEX 미국반도체 ──┤── 상관계수 0.85~0.95        │
│  ACE 빅테크TOP7  ──┤   (하락장에서 → 1.0 수렴)    │
│  ACE 미국S&P500  ──┘                              │
│  KODEX AI전력인프라 ──── 간접 상관 0.70~0.80      │
│  ACE S&P500채혼합 ──── 절반은 테크 블록과 동일    │
│                                                   │
├─────────────────────────────────────────────────┤
│  🟡 방어 블록 (~15%)                              │
│                                                   │
│  ACE KRX금현물 ──── 유일한 비상관 자산            │
│  ACE S&P500채혼합의 채권 50% ── 금리 역상관       │
└─────────────────────────────────────────────────┘
```

NVDA 한 종목이 나스닥100, 반도체, 빅테크TOP7, S&P500 4개 ETF에 모두 포함.
**7개 ETF를 보유하고 있지만, 실질 분산도는 2~3종목 수준.**

### 이 포트폴리오를 움직이는 3개의 힘

| 힘 | 방향 | 포트폴리오 영향 |
|----|------|---------------|
| **USD/KRW 환율** | 원화 강세(↓) | 모든 미국 ETF 손실 (환헤지 없음) |
| **미국 금리 (10Y)** | 금리 상승(↑) | 테크(장기듀레이션) 할인율↑ → 하락 |
| **AI 설비투자 사이클** | 기대 하향 | 반도체+빅테크+AI전력 동시 하락 |

나머지 지표(RSI, MACD, Fear&Greed, VIX 등)는 이 3가지의 **파생물**이다.

### 최악 시나리오

| 시나리오 | 예상 손실 | 확률 |
|---------|----------|------|
| 테크 조정 + 원화 강세 동시 | -20~28% | 중 |
| 광범위 매도 + 달러 강세 | -10~15% | 중 |
| 테크 → 가치주 로테이션 | -5~12% | 높 |
| 인플레 재발 → 금리 인상 | -10~18% (수개월) | 낮 |

## 새로운 설계 원칙

### 이전 설계(v2)와의 차이

| v2 설계 | v3 설계 |
|---------|---------|
| 7개 종목 개별 분석 | **블록 단위 관리** (테크/방어) |
| 매일 동일한 포맷 | **평시 vs 경보 vs 주간** 3모드 |
| 모든 지표 나열 | **3개 핵심 변수만** (환율, 금리, AI사이클) |
| AI가 긴 분석문 작성 | **AI는 변화가 있을 때만** 발언 |
| 보유 종목 밖 기회 | **보유 포트폴리오 관리에 집중** |
| 매일 Action 강제 | **"오늘 할 일 없음"이 가장 좋은 Action** |

### 행동재무학 기반 4원칙

1. **평시에는 조용하게**: 정상인 날은 짧은 메시지. 불필요한 정보는 과잉거래를 유발
2. **이상 시에만 크게**: 임계값 돌파 시에만 경보. 매일 경보하면 경보가 아니다
3. **블록 단위 사고**: 개별 종목이 아닌 "테크 블록 85% → 적정한가?"로 사고
4. **비행동이 최선**: 개인 투자자의 최대 적은 과잉거래. 리포트가 거래를 부추기면 안 됨

## 3모드 리포트 시스템

### 모드 1: 평시 리포트 (1개 메시지)

**발동 조건**: 모든 지표가 정상 범위
**목표**: 3초 만에 "오늘은 괜찮다" 확인

```
📅 2026.03.14 (금) 08:00

🟢 평시 | 포트 수익률 전일 +0.3%

테크블록 +0.4% | 방어블록 +0.1%
USD 1,452(-3) | 미10Y 4.12 | VIX 16.2

전종목 정상 범위. 금일 거래 불필요.
```

**이것이 전부다.** 정상인 날에 7개 종목을 하나하나 보여줄 이유가 없다.
"괜찮다"는 확인만 주고 끝.

### 모드 2: 경보 리포트 (2개 메시지)

**발동 조건** (하나라도 해당 시):
- 테크 블록 일간 변동 ±3% 이상
- VIX 25 이상 또는 전일 대비 +20%
- USD/KRW 일간 변동 ±15원 이상
- 미 10Y 금리 4.5% 돌파 또는 일간 ±0.10%p 이상
- 개별 종목 RSI 75+ 또는 25- 진입
- Mag-7 실적/가이던스 서프라이즈 (AI capex 관련)
- WGBI 편입/환율정책 관련 뉴스
- Fear&Greed 20 이하 (Extreme Fear) 또는 80 이상 (Extreme Greed)

#### 메시지 1: 경보 + 상황

```

🔴 경보 | NVDA -5.2% → 테크블록 타격 예상

━━ 무슨 일인가 ━━

엔비디아 실적 가이던스 하향 → AI 설비투자 축소 우려
→ 필라 반도체지수 -3.8%, NQ선물 -2.1%
→ 내 포트: 반도체 ETF 직격, 나스닥·빅테크·AI전력 연쇄 하락 예상

━━ 내 포트폴리오 영향 ━━

테크블록(85%): 🔴 -2.5~4.0% 예상
  KODEX반도체  직격 (-4~6%)  ← AI capex 직접 연동
  KODEX나스닥  연쇄 (-2~3%)
  ACE빅테크    연쇄 (-2~3%)
  KODEX AI전력 연쇄 (-3~5%)  ← AI capex 간접 연동

방어블록(15%): 🟢 완충 작동
  ACE KRX금  +0.5~1% (안전자산 수요)
  S&P채혼합  채권 부분 소폭 상승

총 예상 손실: -2.0~3.5%
```

#### 메시지 2: 판단 + 행동

```
━━ 판단 ━━

이번 하락의 성격:
• 일시적 조정 vs 추세 전환?
  → 가이던스 하향 폭이 -5% 이내면 기술적 조정 가능성 높음
  → HBM 수요 자체는 유지 중 (SK하이닉스 수주잔고 확인 필요)

• 테크블록 85%가 적절한가?
  → 현재 수준에서 추가 -10% 하락 시 포트 전체 -8.5%
  → 감내 가능한가?

━━ 행동 옵션 ━━

A. 아무것도 안 한다 (장기 보유 관점)
   → 조건: AI capex 축소가 -10% 이내, 환율 안정

B. 테크블록 10% 축소 → 금/KOFR 이동
   → 반도체 ETF 20% 매도 @ 37,500원 이상
   → ACE KRX금 또는 TIGER KOFR 매수
   → 테크블록 85% → 75%

C. 손절 라인 설정 후 관망
   → KODEX반도체 35,000원(-12%) 이탈 시 전량 매도
   → 나스닥100 17,000원(-8%) 이탈 시 50% 매도

💡 B 또는 C 중 하나를 장 시작 전에 결정할 것.
   결정을 미루는 것이 가장 나쁜 선택.
```

**v2와의 핵심 차이**:
- "매도 @ 39,200원" 같은 단일 정답 제시가 아니라 **옵션 A/B/C를 제시**
- 각 옵션의 **전제 조건**을 명시 → 투자자가 자기 상황에 맞게 선택
- **"아무것도 안 한다"가 항상 첫 번째 옵션** → 과잉거래 방지

## AI 프롬프트 설계 (v3)

### 핵심 변경: AI의 역할 재정의

**v2**: "매일 투자 인사이트를 생성하라"
**v3**: "변화가 있을 때만, 그 변화의 의미를 해석하라"

### Compact 프롬프트 (평시용 — 경보 발동 시에만 생성)

```

  → 이 5개는 상관계수 0.85~0.95. 하나가 빠지면 전부 빠진다.

3. AI 설비투자 사이클 (둔화 → 반도체+AI전력 동시 하락)
```

### main.py 변경 개요

```python
# Step 0~4: 기존 데이터 수집 (변경 없음)

# Step 4.5: 크로스 프로젝트 데이터 로드 (경보/주간 모드에서만 사용)
from src.cross_project_data import get_enriched_data_for_ai

# Step 5: 경보 판정
from src.alert_engine import AlertEngine
engine = AlertEngine()
mode = engine.determine_mode(all_data)

# Step 6~8: 모드별 분기
if mode == 'normal':
    # AI 호출 없이 1메시지 생성
    messages = [engine.generate_normal_message(all_data)]

elif mode == 'alert':
    # AI 호출하여 상황 해석 생성
    enriched = get_enriched_data_for_ai()
    collected_data += enriched if enriched else ""
    compact, _, _ = researcher.generate_briefing(collected_data)
    messages = engine.generate_alert_messages(all_data, compact)

elif mode == 'weekly':
    # AI 호출하여 주간 분석 생성
    enriched = get_enriched_data_for_ai()
    collected_data += enriched if enriched else ""
    _, detailed, _ = researcher.generate_briefing(collected_data)
    messages = engine.generate_weekly_messages(all_data, detailed)

# 텔레그램 발송
for msg in messages:
    notifier.send_message(msg)
```
