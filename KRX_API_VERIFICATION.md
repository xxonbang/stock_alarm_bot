# KRX API 검증 리포트

## 검증 일시
- 검증 날짜: 2026년 1월 14일
- 인증키 유효기간: 2026/01/13 ~ 2027/01/12

## 인증키 상태
- ✅ 인증키 존재 확인
- ✅ 인증키 길이: 40자
- ✅ 인증키 유효기간 내 (유효)

## API 호출 테스트 결과

### 테스트한 엔드포인트
- `https://data-dbg.krx.co.kr/svc/apis/sto/stk_bydd_trd`
- `http://data-dbg.krx.co.kr/svc/apis/sto/stk_bydd_trd`

### 테스트 결과
- ❌ **모든 호출에서 401 Unauthorized 오류 발생**
- 응답 메시지: `{"respMsg":"Unauthorized API Call","respCode":"401"}`

### 테스트한 파라미터 조합
1. `basDd`만 사용 (전체 종목 조회)
2. `basDd` + `isuCd` (개별 종목 조회)
   - 종목 코드 형식: `005930`, `KR7000010008`, `KR0005930000`

## 원인 분석

### 1. API 서비스 이용 신청 미완료 (가능성 높음)
KRX OpenAPI는 **인증키 발급과 별도로 각 API 서비스에 대한 이용 신청**이 필요합니다.

**필요한 절차:**
1. ✅ 인증키 발급 완료 (2026/01/13)
2. ⚠️  **"유가증권 일별매매정보" API 서비스 이용 신청** (확인 필요)
3. ⚠️  관리자 승인 대기 (1-2일 소요)

### 2. 확인 방법
1. KRX Data Marketplace 로그인: https://openapi.krx.co.kr
2. 마이페이지 > 이용현황 이동
3. "유가증권 일별매매정보" API 서비스 상태 확인
   - 상태가 "승인"인지 확인
   - "대기" 또는 "신청" 상태면 승인 대기 중

## 현재 구현 상태

### ✅ 완료된 작업
1. `config/settings.py`에 `krx_api_key` 추가 (optional)
2. `src/crawler.py`에 `get_kr_stock_data_krx_api()` 함수 구현
3. `get_kr_stock_data()`에 Fallback 메커니즘 추가
4. 기존 함수 시그니처 및 반환 형식 유지
5. 예외 처리 및 타임아웃 설정 (5초)

### ✅ Fallback 메커니즘
- KRX API 실패 시 자동으로 네이버 크롤링으로 전환
- 기존 기능에 영향 없음
- 성능 저하 없음 (2.32초, 5초 이내)

## 참고사항

```

Method: GET
Headers: {
    "AUTH_KEY": "발급받은_인증키"
```

## 추가 API 신청 필요 여부 분석

### 현재 신청한 API
1. ✅ **유가증권 일별매매정보** (신청 완료)
2. ✅ **ETF 일별매매정보** (신청 완료)

### 필요한 데이터 vs 제공 API

#### 1. 외국인/기관 순매매량
- **필요**: 외국인 순매매량, 기관 순매매량 (만 주)
- **제공 API**: ✅ **유가증권 일별매매정보**
  - 컬럼명: `FRGN_NTBY_QTY` (외국인 순매수량), `ORG_NTBY_QTY` (기관 순매수량)
  - 또는: `외국인순매수`, `기관순매수` 등
- **결론**: ✅ **추가 API 불필요** (유가증권 일별매매정보에 포함됨)

#### 2. ETF 괴리율
- **필요**: ETF 괴리율 (NAV 대비 %)
- **제공 API**: ✅ **ETF 일별매매정보**
  - NAV 및 시장가 정보 포함 예상
- **결론**: ✅ **추가 API 불필요** (ETF 일별매매정보에 포함될 가능성 높음)

### 추가로 고려할 수 있는 API (선택사항)

#### 1. 코스닥 일별매매정보
- **필요성**: ⭐⭐ (코스닥 종목이 포트폴리오에 있는 경우)
- **현재**: 유가증권(KOSPI)만 신청됨
- **권장**: 코스닥 종목이 있다면 신청 고려

#### 2. 지수 정보 API
- **필요성**: ⭐ (시장 전체 흐름 파악용)
- **현재**: 미신청
- **권장**: 선택사항 (현재 프로젝트에서 필수 아님)

### 최종 권장사항

**필수 API (현재 신청 완료):**
- ✅ 유가증권 일별매매정보
- ✅ ETF 일별매매정보

**추가 신청 고려 (선택사항):**
- ⚠️  **코스닥 일별매매정보** (코스닥 종목이 포트폴리오에 있는 경우)
  - 예: `000660.KS` (SK하이닉스)는 코스피이므로 불필요
  - 코스닥 종목이 있다면 신청 권장
